Image Overlay Editor

How To Use
  • Choose a background source: Upload Image or Generate Gradient.
  • If you use Upload Image, select your background image from your device.
  • If you use Generate Gradient, create a gradient background and optionally add an overlay image.
  • Click a text block to select it, drag it to move, and double click it to edit.
  • Use Add Layer to create multiple text layers and switch between them from the list.
  • Enable Stroke if you want outline styling for text and overlay image.
  • You can drag the overlay image and resize it from the bottom-right corner.
  • Set custom export width and height before downloading if needed.
  • Download saves the final image and also adds it to Saved Versions below.
